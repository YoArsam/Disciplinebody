# Accountability App (iPhone Ready)

This is your habit accountability MVP. It runs on the web and as a real iOS app via Xcode (using Capacitor).

## What’s inside

- React (UI) + Vite (build)
- Tailwind (styling)
- Capacitor (wraps the web app into a native iOS app)

## Run on your iPhone (Xcode)

1) Install dependencies

```
npm install
```

2) Build the web app and copy it to iOS

```
npm run build:ios
```

If this is your first iOS setup, add the iOS platform once:

```
npm run ios:add
```

3) Open Xcode

```
npm run ios:open
```

4) In Xcode

- Select a Team under Signing & Capabilities
- Choose your iPhone (plugged in) or a Simulator
- Press the Run ▶ button

That’s it — the app launches on your iPhone.

### Optional: live reload while coding

1) Start the dev server on your Mac (LAN accessible):

```
npm run dev:ios
```

2) Edit `capacitor.config.json` to point the iOS WebView to your Mac’s LAN URL and allow HTTP in dev:

```json
{
  "appId": "com.accountability.app",
  "appName": "Accountability",
  "webDir": "dist",
  "bundledWebRuntime": false,
  "server": {
    "url": "http://YOUR_LAN_IP:5173",
    "cleartext": true,
    "iosScheme": "capacitor"
  }
}
```

Replace `YOUR_LAN_IP` with your Mac’s IP (same Wi‑Fi as the iPhone). Then run:

```
npm run cap:sync && npm run ios:open
```

Run from Xcode again. Changes in the code will refresh instantly on the device.

## Project structure (plain English)

- `src/App.jsx` — the app’s brain (state + which screen to show)
- `src/screens/` — easy-to-understand screen names
  - `HomeScreen.jsx` — today view with your habits and Done button
  - `HabitsScreen.jsx` — manage habits (add/edit/delete)
  - `EditHabitScreen.jsx` — the habit editor page
  - `EditWalletScreen.jsx` — change your wallet balance
  - `EditSkipCostScreen.jsx` — change the penalty per miss
- `src/components/` — the building blocks used by the screens
- `src/index.css` — styles (already tuned for iPhone safe areas)
- `index.html` — the web entry point
- `capacitor.config.json` — iOS app wrapper settings
- `ios/` — generated by Capacitor (the Xcode project lives here)

## Useful scripts

- `npm run dev` — run locally in the browser
- `npm run dev:ios` — dev server accessible from iPhone (LAN)
- `npm run build` — production web build
- `npm run build:ios` — build and copy to iOS app
- `npm run ios:add` — add the iOS platform (first time only)
- `npm run cap:sync` — sync web files to native projects
- `npm run ios:open` — open Xcode

## Notes

- The app stores data on the device (no login). Deleting browser data will reset.
- For App Store submission later, we’ll add icons/splash via `npx @capacitor/assets generate --ios`.
